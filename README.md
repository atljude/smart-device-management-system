# smart-device-management-system
 A Python  application that manages smart devices in a smart home.

This project is a Smart Device Management System developed using Python. It demonstrates the concepts of Object-Oriented Programming (OOP) by managing different smart home devices through a menu-driven program.

The system uses inheritance, encapsulation, constructors, methods and classes to perform different operations on smart devices.



The project was developed to demonstrate:

- Classes and Objects
- Inheritance
- Encapsulation
- Constructors
- Methods
- Getters and Setters
- Use of `super()`
- Loops
- Conditional Statements
- User Input and Output

The program allows the user to:

- Display device information
- Turn devices ON
- Turn devices OFF
- Read temperature
- Increase or decrease light brightness
- Start and stop camera recording
- Exit the program

## How to Run the Program

1. Download or clone the project.
2. Open the project folder.
3. Open the terminal or command prompt.
4. Run the following command:

```bash
python smart_device_management.py
```

5. Follow the menu displayed on the screen.

This project demonstrates how Object-Oriented Programming can be used to develop a simple smart home management system. It also shows the use of inheritance and encapsulation to organize and protect data while allowing different smart devices to perform their specific functions.
